# Wiki桌面客户端
一个`非官方`的wiki百科桌面应用程序

## 功能
> 注意！`https://www.wikipedia.org/`在中国是不可访问的

- 查询wiki百科
- 百科内容跳转
- 回车快捷查询
- 精美用户界面
